HTML

简介

超文本标记语言（英语：HyperText Markup Language，简称：HTML）是构建万维网的基础技术，通过标记标签描述网页内容的结构和语义。最新版本HTML5引入了语义化标签和多媒体支持，显著提升了网页的可访问性和交互性。

核心元素

文档结构

<!DOCTYPE>: 声明文档类型为HTML5

<html>: 根元素，包含lang属性定义文档语言（例：lang="zh-CN"）

<head>: 包含元数据和资源链接

<title>: 显示在浏览器标签页的标题

<body>: 包含所有可见内容

内容区块

<header>: 页面头部（HTML5语义化标签）

<nav>: 导航链接容器（HTML5）

<main>: 页面主要内容（HTML5）

<section>: 主题分组容器（HTML5）

<article>: 独立内容区块（HTML5）

<aside>: 侧边栏内容（HTML5）

<footer>: 页面底部（HTML5）

文本格式

<h1>~<h6>: 六级标题体系（注意SEO最佳实践）

<p>: 段落容器，建议每个段落单独使用

<br>: 强制换行（注意与CSS的white-space属性区分）

<hr>: 水平分隔线（HTML5中为语义化元素）

多媒体

<img>: 图像嵌入（需配合alt属性提升可访问性）

<audio>: 音频播放（HTML5新增，支持多种格式）

<video>: 视频播放（HTML5新增，支持controls属性）

<canvas>: 动态绘图（HTML5，通过JavaScript操作）

表单元素

<form>: 表单容器（action/method属性定义提交）

<input>: 多功能输入（type属性支持text/password/email等15+类型）

<button>: 可点击按钮（优于input type="button"的语义化选择）

<select>: 下拉选择框（配合<option>使用）

<textarea>: 多行文本输入

重要属性

全局属性

class: 定义CSS类选择器（可多类共存，例：class="btn primary"）

id: 唯一标识符（用于页面内锚点跳转和CSS/JavaScript操作）

style: 内联CSS样式（优先级高于外部样式表）

title: 悬停提示文本（辅助可访问性）

lang: 定义元素语言（例：lang="en"）

链接相关

href: 指定链接目标（支持相对/绝对路径和锚点）

target: 控制打开方式（_blank新标签页，_self当前页）

rel: 定义链接关系（例：nofollow/noopener/noreferrer）

download: 强制下载资源（需服务器配合Content-Disposition）

表单增强

required: 必填字段验证（HTML5内置验证）

placeholder: 输入框占位提示

autocomplete: 自动填充控制（on/off）

pattern: 正则表达式验证（需配合JavaScript使用）

最佳实践建议

  1. 优先使用语义化标签提升可访问性
  2. 为所有图片添加有意义的alt文本
  3. 使用ARIA属性增强复杂组件的可访问性
  4. 表单字段添加label标签并通过for属性关联
  5. 避免使用已废弃的标签（如<font>、<center>）
